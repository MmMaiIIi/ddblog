Problem set 1

动态数组

摊销成本O(1) （讨论实际情况

正常数组 添加新元素 O(n) 转移到新的内存，重开时间>>添加次数
动态数组 添加新元素 O(1) 转移到新的大(按比例)内存，重开时间匹配添加次数

Problem set 2

区间合并

分治 抽象出本质

Problem set 3

哈希

u >> n 基于链表

通用哈希

初始化时从一组哈希函数随机选择一个特定哈希函数，并在整个生命周期不变

不是每一次查找/插入都更换哈希函数

但是保证不会在特定输入集合下发生collision
