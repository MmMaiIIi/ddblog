Computer architecture Lab2

1.Pipeline basics I

Load Byte (LB) Instruction in Pipeline Stages:

  1. IF Fetch the LB instruction from memory.
  2. **DI ** Determine the opcode (LB) and extract the operand specifiers (rs, rt) from the instruction.
  3. CO Calculate the effective address of rs and rt.
  4. **FO ** Fetch the byte from the main memory at the calculated effective address.
  5. EI Perform the execution phase for the LB instruction.
  6. WO Store the result in memory, typically the content of the fetched byte, into the destination register rt.

ADD (Addition) Instruction in Pipeline Stages:

  1. **IF ** Fetch the ADD instruction from memory.
  2. DI Determine the opcode (ADD) and extract the operand specifiers (rs, rt, rd) from the instruction.
  3. CO Calculate the effective address using the source registers (rs, rt).
  4. **FO ** For ADD instruction, this stage is empty.
  5. **EI ** Perform the addition operation using the values in the source registers (rs, rt).
  6. WO Store the calculated sum into the destination register rd.
