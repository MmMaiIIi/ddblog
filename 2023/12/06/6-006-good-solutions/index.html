Problem set 3

哈希

u —-> n （u映射到n，u 远大于 n

通用哈希

初始化时从一组哈希函数随机选择一个特定哈希函数

后面每一次查找/插入都不会更换哈希函数

初始化的随机性保证不会在特定输入集合下发生collision

Problem set 4

Little Exercise

​ PROOF: A height-balanced tree is balanced.

Balanced:

​ h = O(log n)

Skew:

​ h(right_node) - h(left_node)

​ the height of its right subtree minus the height of its left subtree (the height of an empty subtree is −1

height-balanced:

​ it’s skew is either −1, 0, or 1.

切披萨 **二维最大前缀和**

一维最大前缀和

max_prefix()，return(max_prefix, max_prefix_key) O(1)

每次对树节点的操作（插入、删除、交换），都按照max_prefix和max_prefix_key的**定义**更新节点

return self.root.max_prefix, self.root.max_prefix_key 即可

如何max_prefix() ---> 二维前缀和呢？

循环x，每次循环都取出y_max_prefix

比较每一轮x对应的y_max_prefix即可

Problem set 7

Wafer Power

$$
O(n^3)
$$

先给所有点顺时针编号 1, … , n

清空全场 -> 摈弃常规前推后思路，而是重视如何做到**隔离**

一维

x[i]: 1~i的max连线

x[i] = x[j] + .. -> j ~ i无法用一维表示，区间需要两维

二维

x[i][j] : j ~ i max连线

m[i][j] = 1 当 i，j 可以连线

每个节点循环 O(n^2) ，遍历j ~ i 内所有连线方案, 每条线都是隔离

  • 1 ~ i ~> n
  • i ~ j ~> 1 （区间慢慢扩大
  • j + 1 ~ k ~> i - 1

x[i][j] = max(x[i][j], m[i][j] + x[j + 1][k-1] + x[k + 1][i - 1])

Effective Campaigning

第i日收米 z(i)，如果选择开设政治演说就可以收米d(i)，但是接下来两天都不能开设政治演说了

general：

分 开/不开 两种情况 => x[i] = max(x[i - 1] + z[i], x[i - 3] + z[i - 2] + z[i - 1] + d[i])

base:

x[0] = max(z[i], d[i])

x[1] = max(x[0] + z[1], z[0] + d[0])

x[2] = max(x[1] + z[2], z[0] + z[1] + d[2])
